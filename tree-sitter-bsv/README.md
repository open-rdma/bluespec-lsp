# tree-sitter-bsv

Note: LLM-led implementation.

A Tree-sitter grammar for Bluespec SystemVerilog (BSV).

What this contains
- Grammar definition: `src/grammar.json`
- Node type definitions: `src/node-types.json`
- Generated parser: `src/parser.c` (generated by tree-sitter)
- Tests: `test/` (example .bsv files used for grammar validation)

Quick start

Prerequisites:
- `tree-sitter` CLI (install via `npm install -g tree-sitter-cli`)

Generate and test the parser:

```bash
cd tree-sitter-bsv
tree-sitter generate    # generate C parser files from grammar.json
tree-sitter test        # run grammar tests (uses files in test/)
```

Using the grammar
- The grammar can be embedded into project-language bindings (node, rust, etc.) by following the Tree-sitter embedding instructions.
- The `parser.c` present is the generated parser; if you edit `src/grammar.json` regenerate with `tree-sitter generate`.

Contributing
